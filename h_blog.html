INTRODUCTION TO HTML

HTML stands for Hyper Text Markup Language HTML is the standard markup language for creating Web pages HTML describes the structure of a Web page HTML consists of a series of elements HTML elements tell the browser how to display the content

HISTORY

History of HTML Created in 1989/1990 Allowed publishing and exchanging of scientific and technical documents Allowed electronic linking of documents via hyperlinks

ADVANTAGES

HTML is Easy to Learn and Use. HTML is very easy to learn and understand. HTML is Free. HTML is supported by all Browsers. HTML is the Most Friendly Search Engine. HTML is Simple to Edit. HTML can Integrate Easily with Other Languages. HTML is Lightweight. HTML is Basic of all Programming Languages
